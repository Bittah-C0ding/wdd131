Ponder - JS Variables and Constants

Variables

Variables are containers for storing data.

  • Declare variables before you use them.
  • The value assigned to a variable can change throughout code.
// declare a variable called num and assign the value 6 to it
let num = 6;
// later in code you can reassign a new value to the variable
num = 10;

Constants

Constants are containers for storing data.

  • Declare constants before you use them.
  • The value assigned to a constant cannot change throughout code.
// declare a const called password and assign the value 'R3d1ng!' to it
const password = 'R3d1ng!';

Scope

A variable or constant has global scope if it is declared outside of any block of code (functions, if statements, loops). It has block scope if declared inside curly braces { }.

let course = "CSE131"; // global scope
if (true) {
    let student = "John";
    console.log(course);  // works fine, course is global
    console.log(student); // works fine, accessed within the block
}
console.log(course); // works fine
console.log(student); // ReferenceError: student is not defined (block scope)

Type Coercion

JavaScript automatically converts types when using certain operators.

const one = 1;
const two = '2';
console.log(one + two); // "12" (string concatenation)
console.log(one * two); // 2 (number multiplication)

Script Tag

Use the <script> tag to link your JavaScript file. Place it in the <head> with the defer attribute, or right before the closing </body> tag.

<script src="ponder.js" defer></script>
